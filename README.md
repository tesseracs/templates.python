# Tesseracs — python template

Starter project for [Tesseracs](https://github.com/tesseracs) chat sessions.

Clone URL: `https://github.com/tesseracs/templates.python`

## Layout

- `main.py` — entry; imports `Greeter` from `greeting.py`.
- `greeting.py` — small class example for multi-file Python.
- `run.sh` — optional `requirements.txt` install, then `python main.py`.
